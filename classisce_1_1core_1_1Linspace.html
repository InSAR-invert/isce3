Return a sub-Linspace over the half-open interval [start, stop).

The behavior is undefined if [start, stop) is not a valid interval (i.e. start > stop) or if start or stop are out of range (i.e. if start >= size() or stop > size()).

[in]startstart position
[in]stopend position (not included in interval)
